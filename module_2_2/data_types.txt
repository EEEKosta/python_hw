# 7. Изменяемые и неизменяемые типы данных

Изменяемые (mutable) типы данных:
Эти типы данных можно изменять после их создания, т.е. вы можете изменять значения,
добавлять новые элементы или удалять существующие.
list, set, dict

Неизменяемые (immutable) типы данных:
Эти типы данных не могут быть изменены после их создания.
int, float, bool, str, tuple, frozenset
Но можно перезаписать их например

text = "Hello"
text = "J" + text[1:]
print(text)  # "Jello"

x = 5
x += 1
print(x)  # 6
